Procedure
  • Hover your cursor over the slider handle and use the up and down arrow keys on keyboard for fine tuning.
  • If fringes are not clear modify the fringe width slightly using Slit width slider (7).
  • Fix the eyepiece at different distances from the slit (100cm, 80cm, 60cm) using slider 1 or textbox (2).
  • Place the vertical cross wire on one of the bright fringe (using slider (3) or textbox (4)) and note the value of the corresponding position from the textbox (4).
  • Now move the cross wire using slider (3) or textbox (4) and note the 10th bright fringe.
